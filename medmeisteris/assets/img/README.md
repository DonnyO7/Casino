# 🖼️ Nuotraukos / Photos — kur ką įkelti

Įkelkite tikras nuotraukas į **šį aplanką** (`medmeisteris/assets/img/`) tokiais
**tiksliais pavadinimais**, kaip nurodyta žemiau. Kai failas atsiras —
svetainėje jis bus parodytas automatiškai (kol failo nėra, rodoma stilinga
vietos žyma su pavadinimu).

> Geriausia: `.jpg`, horizontalios (≈ 1600×1200 px), iki ~400 KB.
> Failų pavadinimai turi sutapti **raidė į raidę** (mažosiomis, su brūkšneliais).
